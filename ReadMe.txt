Installation Instructions(window):

1. Copy or cut the "kbb30" folder into the "htdocs" folder on your computer. (The "htdocs" folder is usually located within your XAMPP installation directory.)

2. Launch XAMPP and start both Apache and MySQL services.

3. Open your web browser (Chrome, Firefox, Safari) and enter the following URL: http://localhost/phpmyadmin

4. In phpMyAdmin, click on "New" to create a new database.

5. Enter "kbbdb" as the database name and click "Create" to set up the database.

6. Once the "kbbdb" database is created, click on the "Import" tab in phpMyAdmin.

7. Click "Choose File" and navigate to the "kbbdb.sql" file located in "htdocs/kbb30/".

8. Scroll down and click "Import" to import the database structure and data.

9. Click on "Structure" to view the database.



Installation Instructions (macOS):

1. Copy or cut the "kbb30" folder and paste it into the "htdocs" folder on your computer. The "htdocs" folder is usually located within your XAMPP installation directory. The typical path is: /Applications/XAMPP/xamppfiles/htdocs/

2. Launch XAMPP by finding the XAMPP application(manager-osx).

3. In the XAMPP Control Panel, click "Manage Servers"

4. Click "Start All" and wait until they all started up.

5. After starting them, click on "Open Web" or manually open your web browser (Chrome, Firefox, Safari).

6. In your web browser, enter the following URL: http://localhost/phpmyadmin

7. In phpMyAdmin, click on "New" to create a new database.

8. Enter "kbbdb" as the database name and click "Create" to set up the database.

9. Once the "kbbdb" database is created, click on the "Import" tab in phpMyAdmin.

10. Click "Choose File" and navigate to the "kbbdb.sql" file located in "htdocs/kbb30/".

11. Click "Go" to import the database structure and data.

12. Scroll down to see the import progress and wait until it completes successfully.

13. Click on "Structure" to view the database.






Accessing the Website:

1. In your web browser, enter the following URL: http://localhost/kbb30/login/login.html

2. You will now be redirected to the KUIH BATANG BURUK ORDERING SYSTEM website.

3. Enter the username and password as provided below to access the admin or customer system.
